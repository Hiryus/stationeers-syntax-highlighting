# Installation

* Download the `ic_udl.xml` file.
* In the "Language" menu, select "Define your language...", then click the "Import..." button.
* Select the `ic_udl.xml` file on your disk.
* Done, enjoy.
* (you can now remove the `ic_udl.xml` file if you like)
